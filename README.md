# 🛡️ AI-Powered Phishing Awareness Analyzer

A modern web-based cybersecurity tool developed using Python and Flask that helps users identify potential phishing emails and messages. The application analyzes text for suspicious keywords, unsafe links, and common phishing indicators, then provides a risk assessment along with security recommendations.

## 📌 Project Overview

Phishing is one of the most common cyberattacks used to steal sensitive information such as usernames, passwords, banking details, and personal data. This project aims to improve cybersecurity awareness by automatically detecting phishing characteristics in emails and messages.

The application features a user-friendly browser interface with modern animations, threat analysis capabilities, and real-time phishing detection.

## ✨ Features

* 🔍 Detects suspicious phishing keywords
* 🌐 Identifies unsafe URLs and links

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/phishing-awareness-analyzer.git
cd phishing-awareness-analyzer
```

### 2. Install Dependencies

```bash
pip install flask
```

### 3. Run the Application

```bash
python phishing_awareness.py
```

The application will automatically open in your default browser.
